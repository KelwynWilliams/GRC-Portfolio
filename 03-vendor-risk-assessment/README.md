# Project 03: Vendor Risk Assessment

**Framework:** NIST SP 800-161 | ISO 27036 | Shared Assessments SIG  
**Scenario:** Redbridge Financial Group TPRM Program (Fictitious)  
**Vendors Assessed:** 3 vendors across Tier 1 (Critical), Tier 2 (High), Tier 4 (Low)  
**Effort:** ~16 hours

---

## What This Project Is

A complete Third-Party Risk Management (TPRM) program including:
- Vendor tiering model (4 tiers based on data access, criticality, and regulatory impact)
- Weighted scoring rubric across 8 risk domains
- Security questionnaire based on Shared Assessments SIG structure
- Full assessments of 3 vendors with Approved, Conditional, and Low-risk outcomes
- Ongoing monitoring procedures and contract requirement frameworks

## Why I Built It

Third-party risk is one of the fastest-growing GRC domains, and accounting
professionals have an underappreciated edge: vendor due diligence, contract review,
financial health assessment, and concentration risk analysis are skills accountants
already have. This project makes that advantage explicit with a GRC lens applied.

## Frameworks & Standards Referenced
- NIST SP 800-161 (Supply Chain Risk Management)
- ISO 27036 (Information Security for Supplier Relationships)
- Shared Assessments Standardized Information Gathering (SIG) Questionnaire
