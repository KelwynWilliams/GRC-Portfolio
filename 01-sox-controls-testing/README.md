# Project 01: SOX Controls Testing Simulation

**Framework:** SOX Section 302 & 404 | COSO 2013 Internal Control Framework  
**Scenario:** Redbridge Financial Group (Fictitious — Financial Services)  
**Scope:** 10 key controls across 5 process areas  
**Effort:** ~20 hours

---

## What This Project Is

A complete SOX Section 404 controls testing program including:
- Risk & Control Matrix (RCM) covering Revenue, AP, Payroll, Financial Close, and ITGCs
- Detailed test procedures with evidence requirements and risk-based sample sizes
- Exceptions log with deficiency severity classification (Material Weakness / Significant
  Deficiency / Control Deficiency)
- CFO-level audit memorandum with management response and remediation commitments

## Why I Built It

SOX compliance is the single largest overlap between my accounting background
and GRC practice. After seven years working with financial controls, reconciliations,
and audit processes, I realized I had been living inside a SOX control environment.
This project translates that embedded knowledge into the language GRC roles expect.

## Frameworks & Standards Referenced
- Sarbanes-Oxley Act Sections 302 and 404(a)
- COSO 2013 Internal Control — Integrated Framework
- PCAOB Auditing Standard 2201
- IIA Risk-Based Sampling Guidelines
